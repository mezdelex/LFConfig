# LF Config

- OS: Windows 11
- Terminal: Windows Terminal
- Shell: PowerShell Core

## Setup

- Install `lf` via `scoop install lf`
- Clone this repository `git clone https://github.com/mezdelex/LFConfig $HOME/AppData/Local/lf`
